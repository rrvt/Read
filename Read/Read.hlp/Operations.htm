App Template Operations

Open File

Opens a database file and displays it in the window. Scroll bar works to see the whole file.

Add a Book to the Database

Supply information to the dialog box and select OK button.

Find Book(s) by Author, Protagonist or Title

Provide a target string to find one or move entries that are found in the author, protagonist or title. The target is chosen by selecting one of the radio round check boxes.

Edit a Book Entry

Select an entry to edit and edit in the dialog box. Select OK button to save.

Delete a Book Entry

Select an entry to delete and delete it from the database. The database must be saved to have effect.

Display Books (All or in Sub Groups)

A drop down menu is shown if the triangle is selected and one of the choices selected will cause a portion of the database to be displayed. If the button itself is selected all the entries are displayed. The scroll bar is active and may be used to view the entire list displayed.

Fix Person Links

There are various links in the database. This function ensures that the links area all correct.

Save Database File

Displays a Save File Dialog and saves the in memory version of the database to the file named.

Print

Will do all the normal things to print the current content displayed in the window.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.
